About the Data

This data was sourced from the V-Dem Institute, which compiles data on democracy around the world from x number of countries on a yearly basis. The data is freely available for social science research.

About this Project

This project was created at the University of Washington for Human-Centered Design and Engineering's Information Visualization class. The collaborators on this project were Megan Brown, Samantha Muscat-Scherr, Camila Proffitt, Ellen Reed, and Spencer Wilkerson.
